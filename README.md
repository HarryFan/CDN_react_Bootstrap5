# React 圖片卡片應用

這是一個使用React和Bootstrap 5製作的圖片卡片應用。

## 功能

- 顯示圖片卡片列表
- 每個卡片都有標題、圖片和鏈接

## 如何使用

1. 下載此代碼。
2. 打開 `index.html` 文件在瀏覽器中。
3. 您將看到圖片卡片列表。

## 依賴項

- React
- Bootstrap 5
- Babel
